PLC Designer Introduction    February 10, 2020
Uxbridge, MAFree
This course has passed.

Online Training Available: As an alternative to attending this in-person training class, participants can take PLC Designer Basics – First Steps and PLC Software Development at Lenze’s Online Training Portal

Prerequisites: Knowledge of IEC61131-3 programming – This class assumes you already know how to program and that you are familiar with IEC61131-3. This class teaches you how to program IEC61131-3 using our PLC Designer software tool.

Required Equipment: The participants are requested to bring a suitable laptop with them to the training course (operating system: Microsoft Windows® 7, 8 or 10 (32/64 bit).

Agenda

  • Introduction to controller 3200 C and p500 and the I/O System 1000
  • Structure of »PLC Designer« V3
  • Standard project with I/O mapping, connecting to the controller
  • Project settings/-download, boot project, source code up-/download
  • Using libraries and the device repository
  • Presentation of variable & data types, POUs (program, function block, function), tasks, persistent & retain variables
  • Using monitoring, variable forces, overwriting, trace, text list, logbook
  • Introduction to IEC61131-3 programming languages, focusing on the FUP languages and structured text
  • Using basic data structures
  • Introduction to visualization with the PLC Designer
  • Practical exercises on the training equipment

Training objectives

The participants will gain technical knowledge on how to apply, program, commission and use the diagnostics options of PLC Designer. They will learn the basics of how to use this product and perform IEC61131-3 programming.

Additional course details

Training duration: 1 day
Target group: Plant and machine manufacturers
Target function: Commissioning; Maintenance and servicing; Machine application
Product: PLC Designer V3.x
